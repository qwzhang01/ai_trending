# 行为准则

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残疾、民族、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何，我们都为社区成员提供不受骚扰的体验。

## 我们的标准

有助于创造积极环境的行为包括但不限于：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括但不限于：

- 使用性暗示语言或图像，以及不受欢迎的性关注或骚扰
- 挑衅、侮辱/贬损评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息，如住址或电子邮箱
- 其他在专业场合中被认为不适当的行为

## 责任范围

项目维护者有责任解释可接受行为的标准，并对任何不可接受的行为采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki编辑、问题或其他贡献，并在适当时暂时或永久禁止任何贡献者参与项目。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共空间代表项目或其社区时的行为。代表项目的行为包括使用官方项目电子邮件地址、通过官方社交媒体账户发布内容，或作为指定代表在在线或线下活动中发言。

## 执行

如遇滥用、骚扰或其他不可接受的行为，请通过 [your-email@example.com](mailto:your-email@example.com) 联系项目团队。所有投诉都将被审查和调查，并会做出被认为必要且适当的回应。项目团队有义务对事件举报人的身份保密。

不遵守或不诚实地执行行为准则的项目维护者可能会面临由项目领导层其他成员决定的临时或永久性后果。
